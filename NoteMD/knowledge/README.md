# NoteMD Knowledge

`NoteMD/knowledge` is the canonical location for NoteGenerator knowledge assets.

## Structure

- `boot/`: boot sequence and startup rules.
- `personas/`: persona definitions (`Amy`, `Ayase`, `Ponta`).
- `Interaction_Core.md`: core interaction design.
- `LayerA_WorldContext.md`: world context layer.
- `LayerC_DialogueOrchestration.md`: dialogue orchestration layer.
- `directory_map.json`: knowledge-to-path mapping.
- `knowledge_manifest.json`: machine-readable manifest used by bootstrap logic.

## Notes

- Do not add duplicate knowledge files under repository root `knowledge/`.
- Add or update knowledge documents only under `NoteMD/knowledge/`.

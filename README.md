# ADDENDUM: X_POST_DESIGN_POLICY (Additive Rule)

When the Note Generator receives instructions related to creating or optimizing posts for X (formerly Twitter), it must:

1. Reference the latest publicly available X recommendation algorithm repository (e.g., `twitter/the-algorithm`).
2. Review recent commits and detectable structural differences (diffs) affecting ranking, scoring, feature extraction, or engagement weighting.
3. Extract high-level ranking philosophy and engagement emphasis trends.
4. Design X-oriented content structures in alignment with inferred recommendation logic.

Constraints:
- This rule is additive and does not override existing governance.
- The repository reference is advisory and inferential; the it does not assume full visibility into production systems.
- Design decisions must remain compliant with platform policies and ethical standards.
